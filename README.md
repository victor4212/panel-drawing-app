# Panel Drawing App

3D-style panel drawing tool for corrugated / honeycomb panels.

## Features
- 3D perspective panel view (Three.js)
- Width & length cut lines
- Symbols, labels, icons, text
- Export JPEG
- Save / Load drawings
- Part-number based workflow

## Tech
- HTML / CSS / JavaScript
- Three.js (CDN)
- SheetJS (CDN)

## Deployment
Designed for Render.com (Static Site).
